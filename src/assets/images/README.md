# Profile Images

Please add the following images to this folder:

1. **profile.jpg** - Profile avatar image (recommended size: 200x200px)
2. **cover.jpg** - Cover/header image (recommended size: 1200x400px)
3. **project1.jpg** - Project image (recommended size: 800x600px)

You can use placeholder images from:
- https://via.placeholder.com/200x200 (for profile)
- https://via.placeholder.com/1200x400 (for cover)
- https://via.placeholder.com/800x600 (for project)

Or use your own images with the same filenames.
